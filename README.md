# OpenInput [![NuGet Version](http://img.shields.io/nuget/v/OpenInput.svg)](https://www.nuget.org/packages/OpenInput)
A portable way of accessing HID devices cross-platform

> This project is still in development!

## Supported Platforms

* Windows (RawInput)
* OpenTK

### Supported Devices

* Mouse
* Keyboard
* ~~TouchDevice~~
* GamePad
